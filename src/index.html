Seja bem-vindo ao nosso sistema!

O sistema foi projetado para realizar a coleta de dados de clientes. Você conseguirá cadastrar novos clientes e visualizá-los em uma lista. Todos os dados são registrados em um arquivo .json.

Clicando na opção Visualizar Clientes ao lado, você terá acesso a lista com todos os clientes já cadastrados no sistema.

Clicando na opção Cadastrar Cliente você conseguirá cadastrar um novo cliente.
